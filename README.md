# Void Runner

A brutal sci-fi roguelike text adventure powered by Claude AI. You are alone in deep space — make desperate choices to survive.

## Overview

Each run is procedurally generated by Claude (claude-sonnet-4). No two runs are alike. Most end badly. Permadeath is enabled.

Tone: grim, tense, atmospheric. Think *Alien*, *Dead Space*, *Firefly*.

## Classes

| Class | HP | Shield | Credits | Starting Items |
|---|---|---|---|---|
| **Pilot** | 90 | 30 | 150 | Nav Charts, Emergency Flare |
| **Hacker** | 75 | 20 | 200 | Exploit Kit, Data Scrambler |
| **Soldier** | 120 | 15 | 100 | Combat Armor, Stim Pack |

## Gameplay

- **Choose a class** and enter your designation to begin a run
- Each turn the AI generates a narrative scene and presents 2–4 meaningful choices with real tradeoffs
- You can also type any custom action
- The AI reacts to your class and inventory — a soldier fights differently than a hacker

## Stats

| Stat | Description |
|---|---|
| **Hull Integrity (HP)** | Takes damage in combat. Hits range from -5 to -30. Reaches 0 = death. |
| **Shields** | Absorbs hits. Regenerates slowly in safe areas (+5 to +15). |
| **Credits** | Earned by looting (+20 to +150). |
| **XP / Level** | 150 XP per level. On level-up: +15 max HP, heal 25 HP. |

## Death Screen

On death you see a full run report: designation, class, sectors survived, level reached, total XP, credits, and items carried at time of death.

## Play Online

**[▶ Play on GitHub Pages](https://camouf1age.github.io/Void-runner/)**

You'll need a free Groq API key from [console.groq.com](https://console.groq.com). Your key is stored only in your browser's localStorage — it never leaves your machine except to call Groq's API directly. Groq's free tier is generous (14,000 requests/day).

## Tech Stack

- React (single JSX component)
- Groq API (`llama-3.3-70b-versatile`) for narrative generation — free tier
- Fonts: Orbitron + Share Tech Mono (Google Fonts)
- No external dependencies beyond React and the Anthropic API
